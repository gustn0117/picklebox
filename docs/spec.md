# PICKLEBOX 홈페이지 스펙 (v1)

## 개요
피클볼 라이프스타일 플랫폼 PICKLEBOX의 브랜드 원페이지 스크롤 랜딩.

- 기술: Next.js (App Router) + Tailwind CSS, 정적 export 가능
- 언어: 한국어 중심 + 영문 슬로건 포인트
- 대상: 국내 피클볼 이용자 / 커뮤니티 / 여행객

## 브랜드
- 슬로건: "Open the Box, Play the Joy. / 피클볼로 여는 즐거운 선물상자"
- 보조: "Play, Smile, Connect."
- 컬러: 브랜드 그린(#1FA84E 계열), 밝고 경쾌한 톤
- 심볼: 웃는 피클볼(말풍선) — 로고 자산 사용
